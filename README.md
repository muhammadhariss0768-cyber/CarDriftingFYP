# Car Driving Game

Welcome to the Car Driving Game Unity project! This repository contains the source code, configuration, and essential assets for a mobile-optimized car driving simulation.

## Features
- Realistic car physics and controls
- Mobile-optimized city environment
- Multiple car models and customization
- Firebase integration for analytics and cloud features
- AdMob integration for monetization
- Modular and extensible codebase

## Project Structure
- `Assets/` — Main Unity assets, scenes, scripts, prefabs, and resources
- `Apks/` — Build outputs (APK/AAB files, ignored in version control)
- `keyStore/` — Keystore files for signing builds (ignored in version control)
- `.gitattributes` & `.gitignore` — Configured for Unity and Git LFS compatibility
- `.csproj`, `.sln` — Visual Studio project and solution files

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/muhammadhariss0768-cyber/CarDriftingFYP.git
   ```
2. Open the project in Unity (recommended version: 2021.3 or later).
3. Restore dependencies and packages via Unity Package Manager.
4. Build and run on your target device.

## Version Control & Large Files
- Large binary assets (textures, bundles, keystores, builds) are excluded from git history and tracked via Git LFS or ignored.
- Only main project files, code, and configuration are included for GitHub compatibility.
